Bearsignals / Piko

Stop fighting
your localhost.

Parallel universes for your terminal. Spin up isolated Git worktrees with unique ports and containers instantly.

> go install github.com/bearsignals/piko@latest
System Specifications
01

Context Switching Solved

One command to switch branches, containers, and tmux sessions. No more stashing changes or restarting servers.

02

Port Conflict Immunity

Every environment gets a unique mathematical ID. Run 5 versions of your app simultaneously. Zero collisions.

03

Data Sovereignty

Databases and caches are isolated per environment. Nuke the database in branch A, branch B stays pristine.

04

Agnostic Architecture

Works with your existing docker-compose. Don't use Docker? Piko works in "Simple Mode" for pure binary isolation.
